HTML NEDİR?

HTML, web sayfalarının iskeletini oluşturan bir işaretleme dilidir (markup language); yani bir programlama dili değildir. Görevi, bir web sitesindeki metinleri, resimleri, başlıkları, linkleri, listeleri ve bölümleri tarayıcıya tanıtmaktır. Tarayıcı da bu bilgileri okuyup ziyaretçiye bir web sayfası olarak gösterir. HTML, yapıyı belirler: hangi kısım başlık, hangi kısım paragraf, hangi görsel nereye ait, sayfa nasıl bölümlere ayrılmış — hepsini HTML düzenler. CSS ise bu yapının görünüşünü güzelleştirir, JavaScript ise sayfaya hareket ve etkileşim ekler. Bu yüzden HTML, bir web sitesinin temel taşı ve her şeyin üzerine kurulduğu ilk adımdır.


Html özelliklerini sıralayacak olursak;

  • HTML bir programlama dili değildir.
  • Web sitesinin iskeleti denilebilir.
  • Etiketler kullanarak içerik tanımlar.
  • Metin, görüntü, video ve ses eklemeyi sağlar.
  • Link oluşturmayı sağlar.
  • Semantik etiketler dediğimiz -header,footer,nav,section- gibi etiketlerle sayfanın anlamlı bölümlere ayrılmasını sağlar.
  • CSS ve JAVASCRIPT ile birlikte kullanılır, ileriye dönük uyumludur, her cihazda ve her işletim sisteminde çalışır.

CSS NEDİR?

CSS (Cascading Style Sheets), web sayfalarının görünümünü ve tasarımını belirleyen bir stil dilidir. HTML sayfanın iskeletini oluştururken, CSS bu iskelete renk, düzen, yazı tipleri, boşluklar, konumlandırma, animasyonlar ve genel tasarım görünümünü ekler. Yani CSS sayesinde bir web sayfasını sade bir metin yığını olmaktan çıkarıp modern, estetik ve kullanıcı dostu bir arayüze dönüştürürsün. Arka plan renkleri, buton tasarımları, kutu düzenleri, responsive (mobil uyumlu) yapılar gibi tüm görsel detaylar CSS ile yapılır. CSS olmadan web siteleri yalnızca siyah-beyaz metinlerden oluşurdu; CSS, web tasarımının kalbidir.


CSS KULLANIM TÜRLERİ
EXTERNAL CSS Ayrı bir CSS dosyası bağlama Düzenli ve profesyonel
INTERNAL CSS style etiketi ile küçük sayfalarda pratik
INLINE CSS style="" içinde hızlı ama profesyonel değil


EN SIK KULLANILAN CSS ÖZELLİKLERİ

  1. RENK VE ARKA PLAN
    • color: yazı rengi
    • back-ground color: arka plan rengi
    • back-ground-image: arka plan resmi

  2. YAZI TİPİ VE METİN
    • font-size: yazı boyutu
    • font-family: yazı tipi
    • font-weight: kalınlık
    • text-align: metin hizalama

  3. KUTULAMA (BOX MODEL)
    • width/height: genişlik/yükseklik
    • padding: iç boşluk
    • margin: dış boşluk
    • border: kenarlık
    • border-radius: köşe yuvarlatma

  4. GÖRÜNÜRLÜK VE KONUMLANDIRMA
    • display: görüntülenme türü
    • position: konumlandırma
    • top/left/right/bottom: konum değerleri
    • z-index: üst üste bindirmede derinlik

  5. FLEXBOX
    • display;flex
    • justify-content: yatay hizalama
    • align-items: dikey hizalama
    • gap: elemanlar arası boşluk

  6. GRİD (GELİŞMİŞ DÜZEN SİSTEMİ)
    • display:grid;
    • grid-template-columns: sütunlar
    • gap: grid boşlukları

  7. TRANSITION/ANIMATION
    • transition: yumuşak geçiş
    • animation
    • transform: döndürme, büyütme

  8. DİĞER ÇOK KULLANILANLAR
    • cursor: mouse imleci
    • opacity: saydamlık
    • overflow: taşma kontrolü

BOOTSTRAP NEDİR?

Bootstrap, web sitelerini hızlı ve modern bir şekilde tasarlamak için kullanılan, hazır CSS ve JavaScript bileşenleri sunan bir frontend framework (ön yüz çerçevesi) dir.


BOOTSTRAP'IN EN ÖNEMLİ ÖZELLİKLERİ
Mobil uyumlu (responsive) yapıyı otomatik sağlar
CSS yazmadan modern tasarımlar elde edersin
Grid sistemi sayesinde sayfayı sütunlara kolay bölersin
Bootstrap = Hazır tasarım + düzen sistemi + komponent kütüphanesi
Büyük bir dokümantasyona sahiptir


BOOTSTRAP NASIL KULLANILIR?

Bootstrap’i HTML dosyana eklemek için CDN bağlantısını kullanabilirsin. Bunun için head etiketinin içine Bootstrap CSS linkini, sayfanın en altına yani body kapanış etiketinin hemen üstüne de Bootstrap JavaScript dosyasını ekleyebilirsin.


JAVASCRIPT NEDİR?

JavaScript, web sitelerini dinamik ve etkileşimli hale getirmek için kullanılan bir programlama dilidir. HTML ve CSS ile birlikte web’in 3 temel teknolojisinden biridir

    JAVASCRIPT NE İŞE YARAR?
  • Butona tıklayınca bir şey olmasını sağlar.
  • Form doğrulama yapar (boş mu, doğru mu).
  • Açılır menü, slider, modal, bildirim gibi öğeleri çalıştırır.
  • Tarayıcıyla iletişim kurar.
  • JS kullanarak oyun, animasyon, saat, hesap makinesi vs. yapılabilir.

JAVASCRIPT'IN ÖNEMİ

Neredeyse tüm modern web sitelerinde JavaScript kullanılır. Front-end Developer olma sürecinde öğrenilmesi zorunlu bir dildir. HTML kullanarak oluşturduğumuz buton gibi yapılar JavaScript olmazsa amacını yerine getiremez. Bu butonlara işlev kazandıran Javascripttir.
